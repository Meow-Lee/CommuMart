# CommuMart
"CommuMart - 커뮤니티 기반 이커머스 플랫폼"

쇼핑과 소통이 함께하는 온라인 마켓플레이스 사용자들이 상품을 구매하고 자유롭게 정보를 공유할 수 있는 통합 플랫폼입니다.

🛒 주요 기능
- 상품 판매 및 구매
- 커뮤니티 게시판 (자유게시판, 상품 후기, 문의사항)
- 사용자 간 정보 공유 및 소통

🛠️ 기술 스택
- Backend: Java, Spring Boot, JPA, MySQL
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Deployment: Docker
